# Issue debugging and reporting

## Issues may arise while using PerovStats (especially while it is still in earlier versions)

- ### Sometimes you will be able to figure out an issue by just looking at an error message in the terminal. If it's something you can fix yourself (e.g. invalid input files given) fix it and then re-run the program.

- ### To report a more complex issue, visit the [GitHub issue creation](https://github.com/AFM-SPM/PerovStats/issues/new/choose) page and describe the problem.

    New features can also be explicitly requested using this link

- ### For more general queries that wouldn't fit the template for a GitHub issue you can email t.allwood@sheffield.ac.uk and I will get back to you as soon as possible.
